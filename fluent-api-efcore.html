EF Core Fluent API

    Có 3 phương pháp cấu hình
  • Model wide configuration (database)
  • Entity Configuration (table)
  • Property configuration

Model-wide configuration

Các thuộc tính Model wide được áp dụng cho toàn bộ model trong database

modelBuilder.HasDefaultSchema("Admin");

ex: Ignore Loại bỏ thực thể.

Entity Configuration

Cấu hình của thực thể (Table) được thực hiện bằng phương thức Entity.

modelBuilder.Entity<Employee>().HasKey(t => t.EmployeeID);

Property Configuration

Cấu hình attributes của property

modelBuilder.Entity<Employee>().Property(b => b.Name).HasColumnName("EmployeeName ");

Navigation Properties

Reference Object (If the relationship is one or Zero-or-One)
In the above example, the Department property in Employee class
Collection (If the relationship is Many)
The Employees Collection in the Department Class

HasMany / WithMany chưa được hỗ trợ trong EF Core (.Net-5_14/11/2021)
